## How to Add Topics

1. Go to repository settings
2. Find "About" section on right side
3. Click gear/settings icon
4. Add topics (up to 30)
5. Save

Or use GitHub CLI:

```bash
# Add topics to a repo
gh repo edit Chocolatsuisse74/ai-sales-closer \
  --add-topic ai \
  --add-topic sales \
  --add-topic agents \
  --add-topic automation \
  --add-topic crm
```
